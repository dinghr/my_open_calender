## 四、交互规范

### 4.1 动画规范

| 场景 | 动画 | 时长 |
|-----|------|------|
| 棘龙呼吸 | 缩放 + 位移 | 3s 循环 |
| 尾巴摆动 | 旋转 | 2s 循环 |
| 眨眼 | Y轴缩放 | 4s 循环 |
| 任务完成 | 属性值弹跳 | 0.3s |
| 积分变化 | 数字滚动 | 0.5s |
| 页面切换 | 淡入淡出 | 0.3s |
| 弹窗打开 | 从下往上滑入 | 0.3s |

### 4.2 色彩规范

| 用途 | 色值 | 说明 |
|-----|------|------|
| 主色 | #667eea | 品牌色 |
| 辅助色 | #764ba2 | 渐变辅助 |
| 攻击力 | #ff6b6b | 红色系 |
| 防御力 | #4ecdc4 | 青色系 |
| 积分 | #f59e0b | 金色 |
| 星星等级 | #fbbf24 | 黄色 |
| 月亮等级 | #a78bfa | 紫色 |
| 太阳等级 | #fb923c | 橙色 |
| 背景 | #f5f5f5 | 页面背景 |
| 文字 | #333333 | 主要文字 |
| 次要文字 | #999999 | 次要信息 |

### 4.3 防焦虑设计规范

- ❌ 不使用红色倒计时
- ❌ 不使用"未完成"高亮
- ✅ 使用"已完成"正向展示
- ✅ 使用模糊时间表述（"今天宜完成"）
- ✅ 未完成项默认折叠
- ✅ 排行榜不比较任务完成情况
