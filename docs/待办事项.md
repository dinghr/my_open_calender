## 📁 文件结构

```
frontend/src/
├── App.vue                    # 主应用入口
├── AppContent.vue             # 主内容组件（含所有页面）
├── api/index.ts               # API 接口定义
└── components/
    ├── BottomNav.vue          # 底部导航
    ├── Spinosaurus.vue        # 棘龙伙伴
    ├── PointsLevel.vue        # 积分等级
    ├── TaskCard.vue           # 任务卡片
    ├── RewardShop.vue         # 奖励商城
    ├── DiscussReward.vue      # 商量奖励
    ├── SmartUpload.vue        # 智能录入
    ├── PasswordModal.vue      # 家长密码验证
    ├── ChildSwitchModal.vue   # 小朋友切换
    ├── DailyPractice.vue      # 每日一练
    ├── TeacherAdvice.vue      # 老师建议
    ├── InviteLink.vue         # 邀请链接
    ├── PrintPreview.vue       # 打印预览
    └── ReadingZone.vue        # 阅读专区

backend/
├── main.py                    # FastAPI 入口
├── routes.py                  # API 路由
├── models.py                  # 数据库模型
├── ai_service.py              # AI 服务
└── data/app.db                # SQLite 数据库
```

## 🚀 运行命令

```bash
# 前端
cd frontend
npm run dev      # 开发模式 http://localhost:5173
npm run build    # 构建生产版本

# 后端
cd backend
python main.py   # 启动 API 服务 http://localhost:8000
```
