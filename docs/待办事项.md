## 🚀 常用命令

### Git 分支管理
```bash
# 查看分支
git branch -a

# 切换分支
git checkout dev
git checkout prod

# 查看分支提交历史
git log --oneline -5
```

### 数据库操作
```bash
# 查看 SQLite 数据库
sqlite3 data/dev.db

# 执行 SQL 查询
sqlite3 data/dev.db "SELECT * FROM students;"

# 进入交互模式
sqlite3 data/dev.db
sqlite> .tables
sqlite> SELECT * FROM students;
sqlite> .quit
```

### 服务管理
```bash
# 查看服务状态
sudo systemctl status ai-study-prod
sudo systemctl status ai-study-test

# 重启服务
sudo systemctl restart ai-study-prod

# 查看日志
sudo journalctl -u ai-study-prod -f
```

### 环境切换
```bash
# 开发环境
export APP_ENV=development
export DB_PATH=data/dev.db

# 生产环境
export APP_ENV=production
export DB_PATH=data/prod.db
```
