### 2. 完成证明区（核心功能）

```
┌─────────────────────────────────┐
│  完成证明                       │
│  ┌─────────────────────────────┐│
│  │                             ││
│  │   [打卡照片/视频]            ││
│  │                             ││
│  │   显示学习/作业内容          ││
│  │                             ││
│  └─────────────────────────────┘│
│  [📷 查看大图] [▶ 播放视频]     │
└─────────────────────────────────┘
```

**设计要点：**
- 图片比例：4:3 或 1:1
- 圆角：16px
- 阴影：轻微投影
- 多图：横向滚动展示
- 视频：带播放按钮封面

**媒体类型：**

| 类型 | 图标 | 说明 |
|------|------|------|
| 单图 | 📷 | 一张照片 |
| 多图 | 📷📷📷 | 最多 9 张，横向滚动 |
| 视频 | ▶ | 最长 60 秒 |
| 音频 | 🎵 | 录音/朗读 |

**交互：**
- 点击图片 → 全屏查看（支持缩放）
- 点击视频 → 播放视频
- 长按图片 → 保存/分享
- 滑动 → 多图切换

**技术实现：**
```vue
<template>
  <div class="proof-section">
    <h3 class="section-title">完成证明</h3>
    
    <div class="media-container">
      <!-- 单图 -->
      <image-viewer
        v-if="proof.type === 'image' && proof.count === 1"
        :src="proof.images[0]"
        :enable-preview="true"
      />
      
      <!-- 多图 -->
      <swiper
        v-else-if="proof.type === 'image' && proof.count > 1"
        :slides-per-view="1.2"
        :space-between="12"
      >
        <swiper-slide
          v-for="(img, index) in proof.images"
          :key="index"
        >
          <image-viewer :src="img" />
        </swiper-slide>
      </swiper>
      
      <!-- 视频 -->
      <video-player
        v-else-if="proof.type === 'video'"
        :src="proof.video"
        :poster="proof.cover"
        :max-duration="60"
      />
    </div>
    
    <div class="media-actions">
      <button class="action-btn" @click="viewFullscreen">
        📷 查看大图
      </button>
      <button class="action-btn" @click="download">
        ⬇️ 保存
      </button>
      <button class="action-btn" @click="share">
        📤 分享
      </button>
    </div>
  </div>
</template>
```

```

│  📊 打卡统计                    │
│  ┌────┬────┬────┬────┐        │
```

**设计：**
- 布局：1 行 4 列
- 卡片：白色背景，圆角 12px
- 数据：大数字（20px，加粗）
- 标签：小字说明（12px，灰色）

**统计项说明：**

| 统计项 | 说明 | 计算方式 |
|--------|------|----------|
| 用时 | 本次打卡用时 | 计时器记录 |
| 连续 | 连续打卡天数 | 连续打卡计数 |
| 本周 | 本周打卡次数 | 本周完成次数 |
| 积分 | 本次获得积分 | 任务积分 + 奖励 |

```

= 总计：5 分
```

**积分动画：**
```css

  0% {

  }
  100% {
    opacity: 0;
    transform: translateY(-20px);
  }

  animation: pointsFloat 1s ease-out;
```

## 🔄 交互流程

### 1. 查看打卡详情

```
用户点击打卡记录
  ↓
加载打卡详情数据
  ↓
显示页面（图片、视频、评论等）
  ↓
用户可操作（评论、分享、点赞）
```

### 2. 家长审核流程

```
家长收到审核通知
  ↓
进入打卡详情页
  ↓
点击"审核"按钮
  ↓
查看完成证明
  ↓
选择：
  - 通过 → 发放积分 → 更新状态
  - 拒绝 → 填写原因 → 通知孩子
```

### 3. 发布评论

```
用户输入评论内容
  ↓
点击发送
  ↓
提交到服务器
  ↓
添加到评论列表
  ↓
通知打卡者（可选）
```

### 4. 分享打卡

```
用户点击"分享"
  ↓
选择分享渠道（朋友圈/群/好友）
  ↓
生成分享卡片（图片 + 文字）
  ↓
调用系统分享
```
