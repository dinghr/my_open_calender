## 三、数据库设计

**说明：** SQLite 环境下 JSON 字段以 TEXT 存储，示例中的 JSONB/GIN 为逻辑表达，可在实现时按 SQLite 方式调整索引。

### 3.1 核心表结构

#### 用户表 (users)
```sql
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    phone VARCHAR(20),
    avatar_url VARCHAR(255),
    nickname VARCHAR(64),
    role VARCHAR(20) DEFAULT 'parent' CHECK (role IN ('admin', 'parent', 'student')),
    password_hash VARCHAR(255),  -- 家长密码
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    is_active BOOLEAN DEFAULT TRUE
);

CREATE INDEX idx_users_phone ON users(phone);
```

#### 登录码表 (login_codes)
```sql
CREATE TABLE login_codes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    code VARCHAR(20) UNIQUE NOT NULL,  -- 格式：FAMILY-XXXX-XXXX
    status VARCHAR(20) DEFAULT 'pending' CHECK (status IN ('pending', 'active', 'used', 'disabled', 'expired')),
    max_uses INTEGER DEFAULT NULL,  -- NULL = 无限次, 1 = 一次性
    current_uses INTEGER DEFAULT 0,
    created_by UUID REFERENCES users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    expires_at TIMESTAMP,  -- 可选过期时间
    approved_at TIMESTAMP,
    approved_by UUID REFERENCES users(id),
    note VARCHAR(255)  -- 备注
);

CREATE INDEX idx_login_codes_code ON login_codes(code);
CREATE INDEX idx_login_codes_status ON login_codes(status);
```

#### 小朋友表 (students)

```sql
CREATE TABLE students (

    hobbies JSONB,  -- 爱好标签数组
    avatar_url VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

CREATE INDEX idx_students_grade ON students(grade);
```

- **基础路径:** `/api/v1`

- **响应格式:**
```json
{
  "code": 200,
  "message": "success",
  "data": {},
  "timestamp": 1710144000
}
```

## 七、安全设计

### 7.1 认证授权
- JWT Token 有效期 2 小时
- Refresh Token 有效期 7 天
- 敏感操作需家长密码验证

### 7.2 数据安全
- 用户手机号加密存储
- 图片访问鉴权
- 儿童数据特殊保护

### 7.3 API 安全
- 请求频率限制
- SQL 注入防护
- XSS 防护

### 7.4 审批机制 ⭐ 新增
- 登录码创建需管理员审批
- 账号绑定需管理员审批
- 审批记录可追溯
