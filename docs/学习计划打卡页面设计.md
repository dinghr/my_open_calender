## 📋 页面元素详解

### 1. 顶部导航栏

```
┌─────────────────────────────────┐
│  ← 2026 年 3 月 21 日       [分享] │
│     9:09                        │
└─────────────────────────────────┘
```

**元素：**
- 左侧：返回按钮 + 日期标题
- 副标题：打卡时间（9:09）
- 右侧：分享按钮

**交互：**

### 2. 数据统计卡片区（核心功能）

```
┌─────────────────────────────────┐
│  📊 数据统计卡片                │
│  ┌────┬────┬────┬────┐        │
│  │今日  │运动  │今日  │今日  │        │
│  │学习  │户外  │任务  │完成  │        │

│  └────┴────┴────┴────┘        │
│  ┌────┬────┬────┬────┐        │
│  │图表  │积分  │行为  │其他  │        │
│  │统计  │成就  │习惯  │      │        │

│  └────┴────┴────┴────┘        │

**设计要点：**
- 布局：2 行 4 列网格
- 卡片：圆角 12px，白色背景
- 图标：彩色图标（每类不同颜色）
- 数据：大数字显示（24px，加粗）
- 标签：小字说明（12px，灰色）

**统计项说明：**

| 统计项 | 说明 | 数据来源 |
|--------|------|----------|
| 今日学习 | 今天学习总时长 | 计时器累计 |
| 运动户外 | 今天运动时长 | 手动打卡/设备同步 |
| 今日任务 | 已完成/总任务数 | 任务列表 |
| 今日完成 | 任务完成率 | 计算得出 |
| 图表统计 | 学习趋势图表 | 跳转详情页 |
| 积分成就 | 获得的积分/勋章 | 积分系统 |
| 行为习惯 | 习惯养成统计 | 打卡记录 |
| 其他 | 更多统计项 | 扩展功能 |

### 3. 学习计划标题栏

```

│  学习计划                       │
│  [✨AI 创建][批量][+ 添加计划] │

| 按钮 | 功能 | 颜色 |
|------|------|------|
| ✨AI 创建 | AI 智能生成计划 | 渐变紫 |
| 批量添加 | 批量导入计划 | 蓝色 |
| + 添加计划 | 手动添加计划 | 蓝色 |

**AI 创建功能：**
```
用户点击"AI 创建"
  ↓
弹出 AI 对话框
  ↓
输入学习目标（如：提高数学成绩）
  ↓
AI 分析当前水平 + 目标
  ↓
生成个性化学习计划
  ↓
用户确认/调整
  ↓
保存到计划列表
```

## 🔄 交互流程

### 1. 创建学习计划

```
用户点击"+ 添加计划"
  ↓
弹出创建表单
  ↓
填写：
  - 任务标题
  - 学科分类
  - 预计时长
  - 重复周期（每天/每周/仅一次）
  - 积分奖励
  ↓
保存 → 添加到计划列表
```

### 2. AI 创建计划

```
用户点击"✨AI 创建"
  ↓
输入学习目标（如：期末数学提高 10 分）
  ↓
AI 分析：
  - 当前水平
  - 目标差距
  - 可用时间
  ↓
生成计划建议：
  - 每日任务
  - 重点知识点
  - 预计时长
  ↓
用户确认/调整 → 保存
```

### 3. 打卡流程

```
用户点击"开始计时"
  ↓
计时器启动
  ↓
学习完成 → 点击停止
  ↓
确认完成（可选上传证明）
  ↓
获得积分
  ↓
更新统计数据
  ↓
日历标记打卡
```

### 4. 积分审核流程（家长）

```
家长收到审核通知
  ↓
查看待审定任务列表
  ↓
点击任务 → 查看完成证明
  ↓
判断：
  - 通过 → 发放积分
  - 拒绝 → 填写原因
  ↓
更新任务状态
```
