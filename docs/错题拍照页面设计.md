### 2. 相机取景预览区

```
┌─────────────────────────────────┐
│  ┌─────────────────────────────┐│
│  │                             ││
```

**设计要点：**
- 比例：4:3（适配大多数手机）
- 边框：圆角 16px
- 背景：深色（突出取景内容）
- 辅助线：九宫格网格（可选开关）

**相机参数：**
- 自动对焦：开启
- 闪光灯：自动
- HDR：开启
- 分辨率：1920×1080（平衡质量与大小）

**技术实现：**
```vue
<template>
```

## 🔄 交互流程

### 1. 进入拍照页面

```
用户点击"拍照录入"按钮
  ↓
请求相机权限
  ↓
权限已授予 → 打开相机预览
权限被拒绝 → 显示提示 + 引导设置
  ↓
显示拍照页面
```

### 2. 拍照流程

```
用户点击拍照按钮
  ↓
相机对焦（0.3s）
  ↓
快门动画 + 声音
  ↓
保存照片到临时目录
  ↓
进入图片编辑页（裁剪/旋转）
  ↓
确认后 → OCR 识别 → 录入错题
```

### 3. 相册选择流程

```
用户点击相册按钮
  ↓
打开系统相册选择器
  ↓
用户选择照片（可多选）
  ↓
返回选择的图片路径
  ↓
进入图片编辑页
  ↓
确认后 → OCR 识别 → 录入错题
```

### 4. 多题连拍流程

```
用户切换到多题模式
  ↓
点击拍照 → 拍摄第 1 张
  ↓
继续点击 → 拍摄第 2、3、4...张
  ↓
点击完成
  ↓
批量预览 + 排序
  ↓
批量 OCR 识别
  ↓
逐题确认录入
```

## 📊 图片处理流程

### 1. 图片预处理

```
拍照完成
  ↓
保存到临时目录
  ↓
图片压缩（质量 80%，≤500KB）
  ↓
EXIF 旋转校正
  ↓
进入编辑页
```

### 2. 图片编辑功能

```
编辑页功能：
├── 裁剪
│   ├── 自由裁剪
│   ├── 比例裁剪（1:1, 4:3, 16:9）
│   └── AI 智能裁剪
├── 旋转
│   ├── 90° 旋转
│   └── 微调角度（±15°）
├── 滤镜
│   ├── 原始
│   ├── 增强（提高对比度）
│   ├── 黑白（文档模式）
│   └── 去阴影
└── 标注
    ├── 画笔
    ├── 马赛克
    └── 文字
```

### 3. OCR 识别

```
确认图片
  ↓
调用 OCR API
  ↓
返回识别结果
  ↓
自动填充题目文本
  ↓
用户确认/编辑
  ↓
保存到错题本
```
