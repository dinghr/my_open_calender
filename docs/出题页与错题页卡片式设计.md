## 🎨 出题卡片组件

### JSX 实现

```jsx
export default function QuestionCard({ question, onAnswer, onHint }) {
  const [selectedAnswer, setSelectedAnswer] = useState(null);
  const [showHint, setShowHint] = useState(false);
  
  return (
    <div className="question-card">
      {/* 题目标题 */}
      <div className="card-header">
        <h3 className="question-title">
          📝 题目 #{question.number}
        </h3>
        <span className="question-tag">{question.subject}</span>
      </div>
      
      {/* 题目内容 */}
      <div className="question-content">
        <p className="question-text">{question.text}</p>
        
        {question.image && (
          <div className="question-image">
            <img src={question.image} alt="题目图片" />
          </div>
        )}
      </div>
      
      {/* 选项（选择题） */}
      {question.type === 'choice' && (
        <div className="options-grid">
          {question.options.map((option, index) => (
            <button
              key={index}
              className={`option-btn ${selectedAnswer === index ? 'selected' : ''}`}
              onClick={() => setSelectedAnswer(index)}
            >
              <span className="option-label">{String.fromCharCode(65 + index)}.</span>
              <span className="option-text">{option}</span>
            </button>
          ))}
        </div>
      )}
      
      {/* 答题区（填空题） */}
      {question.type === 'fill' && (
        <div className="answer-input">
          <input
            type="text"
            placeholder="请输入答案..."
            className="answer-field"
          />
        </div>
      )}
      
      {/* 功能按钮 */}
      <div className="card-actions">
        <button className="action-btn-secondary" onClick={() => setShowHint(!showHint)}>
          💡 提示
        </button>
        <button className="action-btn-secondary">
          📝 草稿纸
        </button>
      </div>
      
      {/* 提示内容 */}
      {showHint && (
        <div className="hint-box">
          <p>{question.hint}</p>
        </div>
      )}
      
      {/* 导航按钮 */}
      <div className="card-navigation">
        <button className="nav-btn" disabled={question.number === 1}>
          ← 上一题
        </button>
        <button className="nav-btn-primary">
          {question.isLast ? '提交 →' : '下一题 →'}
        </button>
      </div>
    </div>
  );
}
```

## 🎨 设计规范总结

### 卡片样式
```css
/* 统一卡片样式 */
.card {
  background: #FFFFFF;
  border-radius: 16px;
  padding: 24px 20px;
  margin: 16px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.08);
}
```

### 按钮类型
| 类型 | 用途 | 样式 |
|-----|------|------|
| 主按钮 | 下一题/提交 | 薄荷绿渐变 |
| 次要按钮 | 提示/草稿纸 | 白色边框 |
| 功能按钮 | 解析/举一反三 | 浅灰背景 |
| 成功按钮 | 已掌握 | 浅绿背景 |
| 危险按钮 | 删除 | 浅红背景 |

### 间距规范
```
卡片间距：16px
卡片内边距：24px 20px
按钮间距：12px
选项间距：12px
```
