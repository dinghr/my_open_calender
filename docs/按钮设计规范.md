## 🎨 按钮类型（4 种）

### 类型 A：卡片操作按钮（右上角）

**使用场景：**
- 卡片右上角的操作入口
- 次要功能的快速入口

**示例：**
```
正在学习              + 添加计划
每日任务量            📝 调整
学习模式              ⚙️ 设置
我的生词本            📋 词表
```

**样式：**
```jsx
<button className="btn-card-action">+ 添加计划</button>
```

```css
.btn-card-action {
  /* 尺寸 */
  padding: 6px 12px;
  min-height: 32px;
  
  /* 颜色 */
  background: #E8F5E9;  /* 浅薄荷绿 */
  color: #26D9AD;       /* 薄荷绿 */
  
  /* 字体 */
  font-size: 14px;
  font-weight: 500;
  
  /* 圆角 */
  border-radius: 8px;
  
  /* 边框 */
  border: none;
  
  /* 布局 */
  display: flex;
  align-items: center;
  gap: 4px;
  
  /* 交互 */
  cursor: pointer;
  transition: all 0.2s;
}

.btn-card-action:hover {
  background: #C8E6C9;
  transform: scale(1.05);
}

.btn-card-action:active {
  transform: scale(0.98);
}
```

**变体：**
```jsx
/* 带图标 */
<button className="btn-card-action">
  📝 调整
</button>

<button className="btn-card-action">
  ⚙️ 设置
</button>

<button className="btn-card-action">
  + 添加计划
</button>
```

### 类型 B：主按钮（大按钮）

**使用场景：**
- 页面主要的行动号召
- 最重要的操作

**示例：**
```
→ 开始今日学习
确认提交
下一步
继续成长之旅
```

**样式：**
```jsx
<button className="btn-primary">
  <span className="button-icon">→</span>
  <span className="button-text">开始今日学习</span>
</button>
```

```css
.btn-primary {
  /* 尺寸 */
  width: calc(100% - 32px);
  margin: 16px;
  padding: 16px 24px;
  min-height: 56px;  /* 至少 56px，易点击 */
  
  /* 颜色 */
  background: linear-gradient(135deg, #26D9AD 0%, #20C997 100%);
  color: #FFFFFF;
  
  /* 字体 */
  font-size: 18px;
  font-weight: 600;
  
  /* 圆角 */
  border-radius: 16px;
  
  /* 边框 */
  border: none;
  
  /* 布局 */
  display: flex;
  align-items: center;
  justify-content: center;
  gap: 8px;
  
  /* 阴影 */
  box-shadow: 0 4px 12px rgba(38, 217, 173, 0.3);
  
  /* 交互 */
  cursor: pointer;
  transition: all 0.3s ease;
}

.btn-primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 6px 16px rgba(38, 217, 173, 0.4);
}

.btn-primary:active {
  transform: translateY(0);
  box-shadow: 0 2px 8px rgba(38, 217, 173, 0.3);
}

/* 图标 */
.button-icon {
  font-size: 20px;
}

.button-text {
  flex: 1;
  text-align: center;
}
```

**变体：**
```jsx
/* 带箭头 */
<button className="btn-primary">
  <span className="button-icon">→</span>
  <span className="button-text">开始今日学习</span>
</button>

/* 纯文字 */
<button className="btn-primary">
  确认提交
</button>

/* 带 emoji */
<button className="btn-primary">
  🌱 继续成长之旅
</button>
```

### 类型 C：次要按钮

**使用场景：**
- 次要操作

- 替代选择

**示例：**
```

取消
返回
```

**样式：**
```jsx
<button className="btn-secondary">明天再做</button>
```

```css

  align-items: center;
```

### 类型 D：快捷操作按钮（图标 + 文字）

**使用场景：**
- 功能快捷入口
- 常用操作快速访问

**示例：**
```
[📚]        [📊]        [🔄]
极速刷题   学习报告   重新学习
```

**样式：**
```jsx
<div className="quick-actions">
  <button className="btn-quick">
    <div className="quick-icon" style={{background: '#E3F2FD'}}>
      📚
    </div>
    <span className="quick-label">极速刷题</span>
  </button>
</div>
```

```css
.quick-actions {
  display: flex;
  justify-content: space-around;
  padding: 24px 16px;
  background: #FFFFFF;
}

.btn-quick {
  /* 布局 */
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 8px;
  
  /* 边框 */
  background: none;
  border: none;
  
  /* 交互 */
  cursor: pointer;
}

.quick-icon {
  /* 尺寸 */
  width: 48px;
  height: 48px;
  
  /* 圆角 */
  border-radius: 12px;
  
  /* 布局 */
  display: flex;
  align-items: center;
  justify-content: center;
  
  /* 字体 */
  font-size: 24px;
  
  /* 背景色根据功能变化 */
  /* 学习类：#E3F2FD（浅蓝） */
  /* 数据类：#FFF3E0（浅橙） */
  /* 复习类：#FCE4EC（浅粉） */
}

.quick-label {
  font-size: 12px;
  color: #666666;
  font-weight: 500;
}
```

**背景色规范：**

| 功能类型 | 背景色 | 示例 |
|---------|--------|------|
| 学习类 | #E3F2FD（浅蓝） | 极速刷题、开始学习 |
| 数据类 | #FFF3E0（浅橙） | 学习报告、统计分析 |
| 复习类 | #FCE4EC（浅粉） | 重新学习、错题复习 |
| 工具类 | #E8F5E9（浅绿） | 设置、调整 |
| 通用类 | #F5F5F5（浅灰） | 其他功能 |

## 📝 按钮文案规范

### 卡片操作按钮
- **格式：** 图标 + 文字（可选）
- **长度：** 2-6 个字
- **语气：** 中性、描述性
- **示例：**
  - ✅ "+ 添加计划"
  - ✅ "📝 调整"
  - ✅ "⚙️ 设置"
  - ✅ "📋 词表"
  - ✅ "查看详情"

### 主按钮
- **格式：** 动词 + 名词（可选箭头）
- **长度：** 4-10 个字
- **语气：** 正向激励、行动号召
- **示例：**
  - ✅ "开始今日学习"
  - ✅ "继续成长之旅"
  - ✅ "确认提交"
  - ✅ "→ 开始今日学习"

### 次要按钮
- **格式：** 简洁描述
- **长度：** 2-6 个字
- **语气：** 中性、平和
- **示例：**
  - ✅ "明天再做"
  - ✅ "换一首"
  - ✅ "取消"
  - ✅ "返回"

### 快捷操作
- **格式：** 图标 + 文字
- **长度：** 4 个字最佳
- **语气：** 功能描述
- **示例：**
  - ✅ "极速刷题"
  - ✅ "学习报告"
  - ✅ "重新学习"
  - ✅ "错题复习"

## 📞 常见问题

### Q: 什么时候用主按钮，什么时候用次要按钮？
**A:** 一个页面只有一个主按钮（最重要的操作），其他操作用次要按钮。

### Q: 卡片操作按钮的背景色太浅怎么办？
**A:** hover 时会加深（#E8F5E9 → #C8E6C9），确保可见性。

### Q: 快捷操作的背景色怎么选择？
**A:** 根据功能类型选择：
- 学习类 → 浅蓝 #E3F2FD
- 数据类 → 浅橙 #FFF3E0
- 复习类 → 浅粉 #FCE4EC
- 工具类 → 浅绿 #E8F5E9

### Q: 按钮文字太长怎么办？
**A:** 
- 卡片操作按钮：限制 6 个字以内
- 主按钮：可以换行，但不超过 10 个字
- 次要按钮：限制 6 个字以内
- 快捷操作：限制 4 个字最佳
