## 📱 界面分析

### 整体风格
- **配色：** 薄荷绿渐变背景（清新、护眼）
- **风格：** 极简、清爽、无压力
- **字体：** 大字号数字显示，清晰醒目

### 核心元素

#### 1. 顶部统计区
```
已打卡天数 💗
   114
```
**特点：**
- 大数字展示成就感
- 爱心 emoji 增加情感化
- 浅色背景突出数字

#### 2. 学习进度卡片
```
我的生词本              📋 词表
━━━━━━━━━━━━━━━━━━━━
在学单词 0              未学单词 225
[════════░░░░░░░░░░░░░] 进度条
```
**特点：**
- 卡片式设计
- 进度条可视化
- 右上角功能入口

#### 3. 学习任务区
```
复习词  0/20

┌──────────────────────────┐
│   → 开始学习             │
│        (大按钮)           │
└──────────────────────────┘
```
**特点：**
- 大按钮，易点击
- 箭头图标引导行动
- 清新绿色，积极正向

#### 4. 底部导航栏
```
📖      📺      ✨      🪐      😊
单词    课程    AI 学    发现    我的
```
**特点：**
- 5 个 Tab，图标 + 文字
- 线性图标，简洁
- 选中状态高亮

### 提示词（可以直接用）

```
Create a mobile home screen for kids learning app

Design reference:
- Clean, minimalist style like Duolingo
- Mint green gradient background (#E0F7FA to white)
- Large numbers for statistics
- Card-based layout
- Big, friendly buttons

Key elements:
1. Top: "Days Streak" with large number (e.g., "12")
2. Center: Animated growth tree emoji 🌳
3. Progress card: "Today's Progress" with progress bar
4. Vocabulary card: "My Vocabulary" with word count
5. Big action button: "→ Start Learning" (mint green)
6. Bottom navigation: 4 tabs (Home, Plan, Wrong Questions, Parent)

Design principles:
- No anxiety-inducing elements
- Positive, encouraging language
- Soft shadows and rounded corners
- Touch-friendly (minimum 44px buttons)
- High contrast for readability
```
