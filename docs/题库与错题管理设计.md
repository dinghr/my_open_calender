## 🎯 核心功能设计

### 功能一：题库录入 ⭐

#### 录入方式

**1. 家长手动录入**
```
小程序端 → 家长专区 → 题库管理 → 添加题目

表单字段：
- 学科（语文/数学）
- 年级
- 单元/章节
- 知识点标签
- 题型（选择/填空/计算/应用题等）
- 题目内容（支持文字 + 图片）
- 答案
- 解析（可选）
- 难度（1-5 星）
- 家长辅导指南（背景知识、常见错误、讲解技巧）
```

**2. 拍照/扫描录入**
```
小程序端 → 拍照 → OCR 识别 → 模型对话 → 意图识别 → AI 分析

流程：
1. 家长拍摄题目
2. OCR 识别题目文本
3. 模型对话确认题目类型与意图
4. AI 分析题型、知识点、难度
5. 家长确认/修改
6. 存入题库
```

**3. AI 生成题目**
```
小程序端 → 选择知识点 → AI 生成

输入：
- 知识点（如"两位数乘法"）
- 难度（1-5）
- 题型（应用题、计算题等）
- 数量（一次生成 5-10 题）

输出：
- 题目内容
- 答案
- 解析
- 家长辅导指南
```

**4. 批量导入（Excel/CSV）**
```
后台管理 → 上传文件 → 映射字段 → 导入

模板格式：
学科，年级，单元，知识点，题型，题目内容，答案，解析，难度
数学，3,两位数乘法，乘法运算，应用题，小明有 5 个苹果...,25,3 步计算，2
```

### 功能二：错题录入 ⭐

#### 录入方式

**1. 从每日一练自动记录**
```
学生完成每日一练 → 答错 → 自动加入错题本

自动记录：
- 题目内容（从题库复制）
- 学生答案
- 错误原因（AI 分析或家长选择）
- 错题日期
```

**2. 拍照扫描录入**
```
小程序端 → 扫描错题 → OCR 识别 → 模型对话 → 意图识别 → 加入错题本

流程：
1. 拍摄错题（练习册、试卷）
2. OCR 识别题目
3. 模型对话确认错题类型与意图
4. 家长选择错误原因
5. 加入错题本
6. （可选）关联到题库中的原题
```

**3. 手动录入**
```
小程序端 → 家长专区 → 错题本 → 添加错题

表单字段：
- 学科
- 题目内容（文字或图片）
- 错误原因（粗心/概念不清/方法错误/未学过/其他）
- 学生答案
- 正确答案
- 错题日期
- 来源（作业/考试/练习）
```

#### 错误原因分类

| 原因 | 说明 | 应对策略 |
|-----|------|---------|
| **粗心** | 看错题、算错数、抄错答案 | 提醒仔细审题，养成检查习惯 |
| **概念不清** | 不理解知识点本质 | 重新讲解概念，用生活例子类比 |
| **方法错误** | 解题方法不对 | 示范正确方法，总结解题步骤 |
| **未学过** | 超出当前学习范围 | 标记为"待学习"，后续安排 |
| **其他** | 无法归类 | 家长备注说明 |

### 功能三：易错度计算 ⭐

#### 计算公式

```
易错度 (error_rate) = 答错次数 / 总作答次数

动态更新：
- 每次学生作答后更新
- 分子：wrong_count++
- 分母：total_count++
- error_rate = wrong_count / total_count
```

#### 易错度分级

| 易错度 | 等级 | 说明 | 处理策略 |
|-------|------|------|---------|
| 0.0-0.2 | ⭐ 简单 | 大部分学生能做对 | 基础练习 |
| 0.2-0.4 | ⭐⭐ 中等 | 部分学生会错 | 正常练习 |
| 0.4-0.6 | ⭐⭐⭐ 较难 | 一半学生做错 | 重点讲解 |
| 0.6-0.8 | ⭐⭐⭐⭐ 困难 | 大部分学生做错 | 专题突破 |
| 0.8-1.0 | ⭐⭐⭐⭐⭐ 极易错 | 几乎都做错 | 深入分析，可能是题目有问题 |

#### 易错度应用

**1. 题目推荐**
```
- 优先推荐易错度适中的题目（0.3-0.6）
- 避免过多极易错题（>0.8），防止挫败感
- 根据学生水平动态调整
```

**2. 家长提醒**
```
- 高易错度题目自动提醒家长："这道题易错度较高，建议陪伴讲解"
- 提供家长辅导指南
```

**3. 举一反三**
```
- 高易错度题目自动生成相似题
- 帮助学生巩固薄弱点
```

### 功能四：错题整理到题库

#### 流程

```
错题录入 → 查重 → 家长确认 → 加入题库

1. 查重：搜索题库中是否有相同/相似题目
   - 文本相似度 > 90%：判定为相同题
   - 文本相似度 70-90%：判定为相似题

2. 如果题库已有：
   - 关联错题记录到原题库题目
   - 更新题库的 wrong_count 和 total_count
   - 重新计算 error_rate

3. 如果题库没有：
   - 家长确认是否加入题库
   - AI 分析知识点、难度、题型
   - 生成答案和解析（如果缺失）
   - 加入题库
```

#### 价值

- 不断积累校本/家庭特色题库
- 记录真实易错数据
- 形成个性化学习资源
