## ✅ 设计要点总结

### 1. 卡片式分组
- 每个功能模块独立卡片
- 卡片之间间距 16px
- 圆角 16px，统一规范

### 2. 4 列网格布局
- 功能图标 4 列排列
- 图标 48x48px
- 间距 16px

### 3. 图标配色规范
- 学习管理：多彩配色
- 备课中心：按学科配色
- 系统设置：灰色统一

### 4. 功能分组清晰
- 学习管理（8 个）
- 备课中心（4 个）
- 系统设置（4 个）

### 5. 顶部用户信息
- 头像 + 名称
- 已绑定小朋友数量
- 卡片样式

## 🚀 v0.dev 提示词

```
Create a mobile "Parent Zone" screen for kids learning app

Design reference: Baicizhan (扇贝单词) profile page

Key elements:
1. Top: Parent avatar + name + "已绑定 2 个小朋友"
2. "学习管理" section: 8 function icons in 4-column grid
   - 绑定管理，学习报告，学习计划，登录码
   - 倾向调整，题库管理，目标设置，学习时间
3. "备课中心" section: 4 function icons
   - 古诗备课，精读备课，数学备课，备课记录
4. "系统设置" section: 4 function icons (gray)
   - 密码设置，通知设置，隐私设置，帮助反馈
5. Bottom navigation: 4 tabs

Icon colors:
- Learning management: Colorful (mint green, blue, purple, orange, pink)
- Prep center: By subject (green, blue, orange, purple)
- Settings: Gray (#F5F5F5)

Design principles:
- Card-based layout
- 4-column grid
- Colorful icons with light backgrounds
- Clean, organized, tool-like
- Easy for parents to manage
```
